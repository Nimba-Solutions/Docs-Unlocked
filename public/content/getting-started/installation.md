# Installation

## Prerequisites

- Node.js 14 or higher
- npm or yarn

## Setup Steps

1. Clone the repository:
```bash
git clone https://github.com/yourusername/docs-unlocked.git
cd docs-unlocked
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

Your documentation site should now be running at `http://localhost:5173`!
