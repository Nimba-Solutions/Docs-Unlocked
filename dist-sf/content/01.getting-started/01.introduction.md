# Introduction

Welcome to **Docs Unlocked**! A powerful, lightweight documentation system built specifically for Salesforce Lightning Web Components.

## What is Docs Unlocked?

Docs Unlocked is a React-based documentation component that runs inside Salesforce orgs. It provides a beautiful, searchable documentation experience with minimal setup required.

## How It Works

Docs Unlocked consists of two main parts:

1. **Lightning Web Component (LWC)** - The Salesforce component that embeds the documentation
2. **React Application** - The documentation viewer that runs inside the LWC

Your content is stored as markdown files in a ZIP StaticResource, and a manifest file automatically generates navigation.

## Quick Start

### 1. Build and Deploy

```bash
npm run build:sf
cci task run deploy --path force-app --org dev
```

This will:
- Compile and bundle the React application
- Package your content files into a ZIP StaticResource
- Generate a navigation manifest
- Deploy everything to Salesforce

### 3. Add to a Lightning Page

1. Open Lightning App Builder
2. Add the "Docs Unlocked" component to your page
3. Configure the component properties:
   - **Content Resource Name**: `docsContent` (or your custom name)
   - **Display Header**: Show/hide the header
   - **Header Label**: Customize header text
   - **Display Footer**: Show/hide the footer

That's it! Your documentation is now live.

## Content Structure

Docs Unlocked uses a simple file-based structure:

- **Sections** are directories (e.g., `01.getting-started/`)
- **Pages** are markdown files (e.g., `01.introduction.md`)
- **Numerical prefixes** control ordering (hidden in the UI)
- **Manifest file** is auto-generated during build

## Navigation

Navigation is automatically generated from your content structure:

- **Sidebar** - Shows all sections and pages
- **Search** - Press `Ctrl+K` (or `Cmd+K` on Mac) to search all content
- **Previous/Next** - Navigate between pages at the top of content
- **Deep Links** - Share links to specific pages or sections
